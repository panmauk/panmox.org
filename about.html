About Us

PANMOX is a small independent project building free and open-source software. Whether it’s a browser, an offline AI, a search engine, a system tool, or something stranger — everything we ship is released under the GNU GPL v3 license. No telemetry. No accounts. No subscriptions.

We believe in transparency, user freedom, and putting control back in your hands. The software you install today should still work in twenty years, on the same hardware, without asking permission from a server somewhere.

Some of our projects are practical. Some are jokes. Some run on machines the rest of the world abandoned. All of them are yours to keep. We currently ship Windows builds, with GNU/Linux and macOS versions on the way.
