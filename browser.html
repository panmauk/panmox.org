PANMOX Browser

Secure browsing, reimagined.

PANMOX is a privacy-first, feature-rich browser built for people who care about security without sacrificing speed or style.

Features
  • Built-in ad & tracker blocker
  • HTTPS-Only mode
  • Fingerprint protection
  • WebRTC leak prevention
  • Third-party cookie blocking
  • Do Not Track & Global Privacy Control
  • Password manager with encrypted vault
  • Address & credit card autofill
  • Cookie consent auto-dismiss
  • Page translation
  • Picture-in-Picture
  • Split view — two sites side by side
  • Tab pinning, muting, hibernation & search
  • Reading list & built-in notes
  • Bookmarks bar
  • Download manager
  • Session restore & crash recovery
  • Multiple profiles
  • Dark & light themes
  • Sci-fi inspired UI
  • Force dark mode on any website
  • Configurable search engine (Google, DuckDuckGo, Bing, Brave)
  • Global media key support (play/pause/skip)
  • Per-tab audio indicator with click-to-mute
  • Site X-Ray — see what technologies a page uses
  • Windows 10 or later (64-bit)
  • 4 GB RAM minimum
  • 300 MB disk space
  • Chromium (via Electron)
